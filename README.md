# README

a) Выбор набора данных  
Для выполнения задачи семантической сегментации был выбран датасет "ISIC 2018 Skin Lesion Segmentation Dataset" (https://challenge.isic-archive.com/data/#2018). Данный набор данных представляет собой коллекцию дерматоскопических изображений кожных новообразований с соответствующими масками сегментации. В датасете содержится более 2500 изображений, размеченных экспертами-дерматологами, что обеспечивает высокое качество аннотаций.  
Обоснование выбора:  
Выбор данного датасета обусловлен его практической значимостью. Сегментация кожных новообразований является важным этапом в системах компьютерной диагностики рака кожи (в частности, меланомы). Автоматизация данного процесса позволяет повысить точность диагностики, снизить нагрузку на врачей, ускорить выявление заболеваний на ранних стадиях. Таким образом, задача сегментации на данном датасете напрямую связана с реальными медицинскими приложениями и имеет высокую социальную значимость.  
  
b) Выбор метрик качества  
Для оценки качества моделей семантической сегментации были выбраны следующие метрики:  
1. Intersection over Union (IoU) - IoU является основной метрикой для задач сегментации и вычисляется как отношение площади пересечения предсказанной и истинной областей к площади их объединения.  
Обоснование:  
Метрика учитывает как ложноположительные, так и ложноотрицательные ошибки, что делает её наиболее информативной для задач сегментации.
2. Dice coefficient (коэффициент Дайса) - данная метрика измеряет степень совпадения между предсказанной и истинной масками.  
Обоснование:  
Особенно эффективна в медицинских задачах, где объекты (например, опухоли) могут занимать небольшую часть изображения. Лучше отражает качество сегментации при дисбалансе классов.
3. Pixel Accuracy - определяет долю правильно классифицированных пикселей.  
Обоснование:  
Позволяет оценить общую точность модели, однако может быть менее информативной при несбалансированных данных, поэтому используется в дополнение к другим метрикам.
